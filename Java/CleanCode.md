# 1. 다른 개발자들을 놀래키지 말자(부디..)
- 새로운 코드를 마주했을 때, 대부분의 개발자는 다음의 감정을 느낀다
- 두려움, 흥분, 놀라움, 분노, 그리고 좌절
- 흔히 *레거시 코드* 라 부르는 코드는 아무도 건드리고 싶지도 않고 모두가 두려워한다
